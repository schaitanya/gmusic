### About
***
beta v0.1
Google Music unofficial API/player written in Node.js

### API
***
Available API Calls

    1. Login (POST /login)
    2. List Albums (GET /albums)
    3. List Songs (GET /songs)
    4. Play Song (GET /song/:id)
    5. Search Song (POST /song)
